# Mis Certificados - Blog de Logros

Este es un sitio web personal para mostrar mis certificados y logros profesionales.

## 📋 Características

- Diseño responsive y moderno
- Filtros por categorías de certificados
- Sección de habilidades con barras de progreso
- Formulario de contacto funcional
- Animaciones suaves
- Navegación intuitiva

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Animaciones)
- JavaScript (Vanilla)
- Font Awesome para iconos
- Google Fonts
